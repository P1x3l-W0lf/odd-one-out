# odd-one-out
A simple game where you need to pick the one emoji that is different in as few tries as possible.

### ✨ Intro
this is a browser-based Odd One Out game made using javaScript, HTML, and CSS so that it is accessible in any modern web interface. There are lots of different emoji spreads, intuitive gameplay mechanics, and your score, click count, and accuracy percetage are displayed at the bottom of the page.

### 📃 Instructions
Open the game in your browser.

There will be a spread of nine emojis, you need to find the odd one out.

Once you have found the imposter, click on it.

You will either move on to the next randomized level if you're correct or be prompted to try again if incorrect.

### ⚙ Configuration
No configuration is required to run the project.

However, if you wanted to modify the game you could...

• Add more emoji combos in the script.js file by adding their hex value into the "emojis" list

• Adjust gameplay logic, win/lose messages, etc. in script.js

• Add more buttons or otherwise adjust the GUI in index.html

• Change the styling of the page in style.css
